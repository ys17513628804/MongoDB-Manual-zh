# [ ](#)聚合命令

[]()

> **注意**
>
> 有关特定命令的详细信息，包括语法和示例，请单击特定命令以转到其参考页面。

| 名称            | 描述                                     |
| --------------- | ---------------------------------------- |
| [`aggregate`]() | 使用聚合框架执行聚合任务，例如分组。     |
| [`count`]()     | 计算集合或视图中的文档数。               |
| [`distinct`]()  | 显示在集合或视图中为指定键找到的不同值。 |
| [`mapReduce`]() | 对大型数据集执行map-reduce聚合。         |

有关不同方法的详细比较，请参阅“ 聚合命令比较”。
