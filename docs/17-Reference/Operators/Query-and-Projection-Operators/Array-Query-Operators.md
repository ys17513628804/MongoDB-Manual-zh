| 名称             | 描述                                                         |
| ---------------- | ------------------------------------------------------------ |
| [`$all`]()       | 匹配包含查询中指定的所有元素的数组。                         |
| [`$elemMatch`]() | 如果array字段中的元素符合所有指定`$elemMatch`条件，则选择文档。 |
| [`$size`]()      | 如果数组字段为指定大小，则选择文档。                         |
