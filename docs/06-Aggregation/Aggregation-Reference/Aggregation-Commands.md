## <span id="id1">聚合命令</span>

| 名称          | 描述                                                 |
| ------------- | ---------------------------------------------------- |
| [aggregate]() | 使用聚合框架执行聚合任务，例如 group。               |
| [count]()     | 计算集合或视图中的文档数。                           |
| [distinct]()  | 显示在集合或视图中为指定 key 找到的不同值。          |
| [mapReduce]() | 对大型数据集执行[map-reduce](../Map-Reduce.md)聚合。 |
